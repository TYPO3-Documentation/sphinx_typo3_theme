Installation¶

Stable release¶

To install Sphinx TYPO3 theme, run this command in your terminal:

$ pip install sphinx-typo3-theme

This is the preferred method to install Sphinx TYPO3 theme, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.
